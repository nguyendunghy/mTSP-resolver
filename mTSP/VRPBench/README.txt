This directory contains VRPBench instances formulated as mTSPs with distance constraints.

The subdirectory INSTANCES contains the following benchmark instances:

	exp (4 instances)
	ManP (30 instances)
	WRPToy (30 instances)

	Sources:
	Müller, F. F, A., Meira, L. A. A.:
	Algoritmos Genéticos Aplicado ao Problema de Roteamento de Veículos.
	Report, arXiv:1808.10866 [cs.NE] (2018)

	Zeni, G. A, Menzori, M., Martins, P. S., Meira, L. L. A.:
	VRPBench: A Vehicle Routing Benchmark Tool.
	Report, arXiv:1610.05402 [cs.AI] (2016)

The instances were generated by PostVRP (available at https://www.ft.unicamp.br/docentes/meira/postvrp/) and converted to LKH-3 format. 

The subdirectory TOURS contains the best tours found by LKH-3.

Tabulated results can be found in the subdirectory RESULTS.
